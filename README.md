# Kana Memory | 線上日語學習工具

一個優雅的日語學習 Web 應用，提供假名、單詞、句子三大學習模式，支援學習方案與成就系統，助你系統化掌握日語。

🌐 **線上體驗**: [kana.yahaha.net](https://kana.yahaha.net/)

## ✨ 主要功能

### 📚 三大學習模式

- **假名學習** - 完整支援清音 46 個、濁音 25 個、拗音 33 個（共 104 個）
- **單詞學習** - 按假名行分類學習日語單詞，支援拗音單詞，智慧篩選
- **句子學習** - 場景化句子練習，涵蓋日常對話、旅遊、餐飲等多個場景

### 📋 學習方案

- **五十音速成** - 7 天系統化掌握平假名與片假名
- **日常會話入門** - 14 天學習常用單詞與基礎句子
- 自動追蹤學習進度，階段化學習目標

### 🏆 成就系統

- **打卡勳章** - 第一步、堅持一週、習慣養成等
- **學習勳章** - 單詞達人、計劃通等
- 解鎖動畫與慶祝效果

### 🎯 學習功能

- **多種顯示模式** - 混合 / 平假名 / 片假名 / 羅馬音 / 互換
- **雙重練習模式** - 學習模式（自動提示）/ 記憶模式（手動提示）
- **智慧 TTS 發音** - 基於 Web Speech API 的標準日語朗讀
- **自動發音選項** - 學習模式下可開啟自動朗讀
- **假名選擇器** - 視覺化選擇清音、濁音、拗音範圍
- **鍵盤快捷鍵** - 全鍵盤操作支援，含 Gmail 風格跳轉（G+H/K/W/P）與 Enter 開始
- **主題模式** - 亮色 / 暗色 / 跟隨系統
- **響應式設計** - 完美適配手機、平板、桌面
- **資料持久化** - IndexedDB 儲存學習進度和偏好設定，支援離線使用

## 🚀 快速開始

### 本地執行

```bash
# 克隆專案
git clone https://github.com/dofy/KanaSyllabaryMemory.git
cd KanaSyllabaryMemory

# 安裝依賴
pnpm install

# 啟動開發伺服器
pnpm dev

# 瀏覽器訪問 http://localhost:3000
```

### 建置部署

```bash
# 建置生產版本
pnpm build

# 預覽建置結果
pnpm preview
```

## 📖 使用說明

### 假名學習

1. 點擊「設定」選擇要練習的假名類型（清音、濁音、拗音）
2. 選擇顯示模式和練習模式
3. 點擊「開始」進入練習
4. 使用鍵盤快捷鍵快速操作

### 單詞學習

1. 在設定中選擇假名範圍
2. 系統自動篩選包含所選假名的單詞
3. 學習模式下自動顯示中文和發音
4. 記憶模式下手動控制提示

### 句子學習

1. 選擇學習場景（日常對話、旅遊、餐飲等）
2. 可多選場景組合學習
3. 支援顯示中文提示和日語發音

### 學習方案

1. 在首頁或學習方案頁面選擇適合的方案
2. 按階段完成學習目標
3. 系統自動記錄進度，支援斷點續學

### 成就系統

1. 完成學習自動打卡
2. 達成條件時自動解鎖勳章
3. 在成就頁面查看所有勳章和進度

## 💡 學習建議

- **初學者**: 選擇「五十音速成」方案 → 系統化學習假名
- **進階者**: 單詞學習 → 按假名行逐步擴展詞彙量
- **熟練者**: 句子學習 → 場景化練習，提升實戰能力
- **記憶強化**: 使用記憶模式 + 互換/羅馬音模式反向練習

## 🛠️ 技術棧

- **框架**: Vite + React 18
- **路由**: React Router v6
- **UI**: Tailwind CSS + shadcn/ui
- **語音**: Web Speech API
- **語言**: TypeScript
- **儲存**: IndexedDB (Dexie.js)
- **部署**: Vercel
